
=head1 NAME

darktable-cmstest - test if the color management subsystem of your computer is correctly configured

=head1 SYNOPSIS

    darktable-cmstest

=head1 DESCRIPTION

B<darktable> is a digital photography workflow application for B<Linux>, B<Mac OS X> and several other B<Unices>.
It's described further in L<darktable(1)|darktable(1)>.

B<darktable-cmstest> investigates if the color management subsystem of your computer is correctly configured and it displays some useful information about the installed monitor profile(s).
